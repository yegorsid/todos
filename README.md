# Todos

Simple tasks app made with [ChakraUI](https://v2.chakra-ui.com/) and [Zustand](https://github.com/pmndrs/zustand)

To run locally: 

```bash
npm install
npm run dev
```

To run test:

```bash
npm run test tests/test.spec
```
